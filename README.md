# Jinmo's Homepage

[https://www.jinmo.kim](https://www.jinmo.kim/)

## Project structure

```
$PROJECT_ROOT
│   # Page files
├── pages
│   # React component files
├── components
│   # Non-react modules
├── lib
│   # Static files for images and 3d model file
└── public
```

## Install & Run
```Bash
npm install
npm run dev
```
